41/* Error Code
42 * If value returned by MultiBootMain() is not 0, an error listed
43 * below has occurred. However, there is no need to have separate error
44 * messages for all error code types.
45 * "Communication Error. Please check connection. Retry with START."
46 * "Turn off power, check connection, and turn on again."
47 * "Slave is not connected."
48 * "Cannot recognize the slave."
